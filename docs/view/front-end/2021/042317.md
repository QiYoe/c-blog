## 1、创建本地项目

### Method 1

打开 [vuepress-theme-reco官方文档](https://vuepress-reco-doc.vercel.app/)，导航到”快速开始“，自己查看文档。。。

当然，觉得麻烦不想看文档的童鞋，可以直接运行：

```bash
  npx @vuepress-reco/theme-cli init
```

![init-project](../../images/042317/init-project.png)

回车! Wait a minute...

下载好之后改改名字配置就可以啦 :smile:
